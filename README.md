# Personal Expense Tracker

A simple Python project to manage and track personal expenses.  
This program allows you to add, view, delete expenses, and see summaries by category. All data is stored locally in a JSON file, making it persistent across sessions.

---

## Features

- Add new expenses with date, amount, category, and description  
- View all recorded expenses  
- Delete expenses by selecting an index  
- View summary of expenses grouped by category  
- Data is saved in `data/expenses.json` for persistence  
- Simple, console-based menu-driven interface

---

## Tech Stack

- **Language:** Python 3.x  
- **Data Storage:** JSON file (`data/expenses.json`)  
- **Libraries:** Standard Python libraries (`json`, `os`)
